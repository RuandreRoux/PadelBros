# 🏓 PadelBros - Tournament Tracker

A mobile-friendly padel tournament tracker for 8 players with random team generation, score tracking, leaderboard, and match history.

## Features

- 🎲 Random team generation (2v2)
- 📊 Live leaderboard
- ✏️ Edit player names
- 📝 Match history with score editing
- 📱 Mobile-friendly design
